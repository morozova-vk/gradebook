Справка

Описание и помощь по работе с приложением

Вкладка «Загрузка» предназначена для импорта данных из внешних файлов.

Три режима загрузки:

  • Список класса — файл содержит только ФИО учеников (без оценок). Формат: одна колонка с ФИО или две — № и ФИО. Требуется указать класс.
  • Оценки одного класса — файл содержит ФИО, предмет и оценки по датам. Формат: ФИО;Предмет;Дата;Оценка. Требуется указать класс и предмет.
  • Общий список — файл содержит полную информацию: Класс;Предмет;ФИО;Дата;Оценка. Если поле "Класс" пустое, ученик ищется по ФИО во всех классах.

Поддерживаемые форматы: .csv, .txt, .xlsx

Даты: поддерживаются форматы ДД.ММ (год подставляется автоматически) и ДД.ММ.ГГГГ.

ФИО: должны быть полными (минимум 2 слова). Допускаются разные пробелы — они автоматически нормализуются.

Вы можете скачать примеры и шаблоны файлов для каждого режима.

Вкладка «Журнал» — основное рабочее пространство для работы с оценками.

Как пользоваться:

  • Выберите класс и предмет из выпадающих списков
  • Таблица отображает: №, ФИО, средний балл и оценки по датам уроков
  • Даты прокручиваются горизонтально
  • Кликните на ячейку с оценкой, чтобы изменить её (2-5 или удалить)
  • Кликните на ячейку с «+» в строке «Тема урока» или «Домашнее задание», чтобы добавить текст

Добавление ученика:

  • Введите ФИО в поле «ФИО ученика»
  • Опционально введите оценки в формате: ДД.ММ=оценка (например: 02.09=5)
  • Нажмите кнопку «+ Добавить»

Добавление даты урока: выберите дату и тему, нажмите «+ Добавить дату»

Управление классом: кнопки «Добавить класс» и «Удалить класс»

Сохранение и экспорт: нажмите «Сохранить» для сохранения в браузере или «Скачать данные оценок» для экспорта в CSV

Вкладка «Статистика» показывает анализ успеваемости в табличной форме.

Два независимых блока:

  • Статистика класса по предмету — выберите класс и предмет, отобразится информация по выбранной группе
  • Общая статистика по предмету (все классы) — выберите только предмет, отобразятся данные по всем классам

Для каждого блока показывается: средний балл, медиана, количество и процент оценок (2-5).

Шкала оценок: 2 (неудовл.), 3 (удовл.), 4 (хорошо), 5 (отлично).

Вкладка «Графики» отображает статистику в виде столбчатых диаграмм.

Два независимых графика:

  • График класса по предмету — выберите класс и предмет
  • Общий график по предмету (все классы) — выберите только предмет

Цветовая кодировка:

  • 🟢 5 (отлично)
  • 🔵 4 (хорошо)
  • 🟡 3 (удовлетворительно)
  • 🔴 2 (неудовлетворительно)

Электронный журнал — система для учёта успеваемости учеников школы.

Основные возможности:

  • Импорт данных из CSV, TXT, XLSX файлов (3 режима)
  • Автоматическая нормализация ФИО (разные пробелы, невидимые символы)
  • Ведение журнала с темами уроков и домашними заданиями
  • Статистика в табличной и графической форме
  • Автосохранение в localStorage браузера
  • Экспорт данных в CSV

Технологии: HTML, CSS (Tailwind), JavaScript, Chart.js, XLSX

Версия: 1.0 (2025)
